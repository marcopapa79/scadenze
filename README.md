# Gestione Scadenze

Applicazione Python per la gestione delle scadenze.

## Setup

1. Crea un ambiente virtuale:
   ```bash
   python -m venv venv
   ```

2. Attiva l'ambiente virtuale:
   ```bash
   # Windows
   venv\Scripts\activate
   
   # Linux/Mac
   source venv/bin/activate
   ```

3. Installa le dipendenze:
   ```bash
   pip install -r requirements.txt
   ```

## Utilizzo

```bash
python main.py
```

## Funzionalità

- [ ] Aggiungere scadenze
- [ ] Visualizzare scadenze
- [ ] Notifiche per scadenze in arrivo
